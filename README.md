# Portfolio

A cleaned Next.js portfolio centered around a single cinematic one-page experience.

## Stack

- Next.js 14
- React 18
- Framer Motion
- Lenis
- Lucide React

## Run

```bash
npm run dev
```

Open `http://localhost:3000`.

## Structure

- `src/app`: app shell, metadata, and global styles
- `src/components/CinematicPortfolio.tsx`: main one-page portfolio
- `src/components/HeroWaterRipple.tsx`: full-hero ripple canvas effect
- `src/components/SmoothScroll.tsx`: Lenis wrapper
- `public/images`: project visuals
- `public/certificates`: certificate assets
